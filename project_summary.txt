LIFECARD SOS - PROJECT SUMMARY & CHANGE LOG

=========================================
1. PROJECT DESCRIPTION
=========================================
LifeCard SOS is an offline-first Emergency Medical ID and SOS tracking application built in Flutter. It securely stores critical user information (allergies, medical conditions, blood group, medications, and primary care contacts) on-device using Hive CE + AES encryption. In an emergency, it facilitates countdown-based alarms, audio sirens, flashlight flashing, geolocated SMS updates, and lock screen widgets.

│   ├── contacts/                  # Emergency contacts viewer and editor
│   ├── home/                      # Tab shell wrapper and dashboard view
│   ├── medical/                   # Detailed medical ID cards viewer and editor
│   ├── onboarding/                # App welcome tutorial and permissions setup flow
│   ├── profile/                   # Basic personal details editor
│   ├── settings/                  # Countdown duration settings, themes, and language preference toggles
│   └── sos/                       # Panic countdown, siren sound, and flashing UI controls
├── models/                        # Serialized schemas with Freezed annotations
├── repositories/                  # DB data access wrappers (profile, settings, helplines, etc.)
├── services/                      # Low-level service tasks (alarm audio, geolocating, local notifications)
└── shared/                        # App-wide Riverpod provider configurations
